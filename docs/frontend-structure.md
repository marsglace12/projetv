# Projet V — Frontend

## Stack
- Vite + React + TypeScript
- React Router (routing)
- Tailwind CSS (styling)

## Structure
- src/app: wiring applicatif (router, layout, providers plus tard)
- src/pages: pages routées du MVP
- src/shared: UI et helpers réutilisables

## Commands
- npm run dev
- npm run build
- npm run preview
- npm run lint
- npm run format
